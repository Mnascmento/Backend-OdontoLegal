# 🦷 OdontoLegal

Quando finalizado, o projeto será um web app completo para **gestão de casos de perícia odontolegal**, permitindo que peritos registrem casos, adicionem evidências, redijam laudos e gerem relatórios organizados.
Neste repositório se encontra o backend do projet, API de perícia criminal odonto-legal.

## 📌 Funcionalidades

- Cadastro e gerenciamento de **casos periciais**
- Adição de **evidências** por caso
- Redação de **laudos periciais** vinculados às evidências
- Extração de **relatórios completos** dos casos
- Interface moderna e responsiva com React
- API robusta e escalável com Node.js e MongoDB (Mongoose)

## 🚀 Tecnologias Utilizadas

### 🧠 Back-End
- [Node.js](https://nodejs.org/)
- [Express](https://expressjs.com/)
- [Mongoose](https://mongoosejs.com/) (ODM para MongoDB)
- [MongoDB](https://www.mongodb.com/)

### 🛠️ Outras Ferramentas
- [Postman](https://www.postman.com/) (para testes de API)
- [Git](https://git-scm.com/) e GitHub para versionamento
